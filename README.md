# Math-Reasoning-Solver
🧮 Text-to-Math Problem Solver (LLM + Streamlit)

A smart AI-powered math problem solver built using Streamlit, LangChain, Groq LLM, Wikipedia API, and NumExpr.
It can solve:

🧮 Mathematical expressions

📚 Conceptual math questions (via Wikipedia + LLM)

🧠 Reasoning-based problems

🚀 Features

✔ Solve direct math expressions (fast computation using numexpr)

✔ Answer theory-based math questions using Wikipedia + LLM

✔ Handle reasoning problems using Groq LLM (Qwen model)

✔ Chat-style interactive UI using Streamlit

✔ Hybrid AI system (Rule-based + LLM + Knowledge retrieval)

🧠 System Architecture

The app follows a 3-layer decision system:

Math Expression Detector

Direct evaluation using numexpr

Knowledge-Based Questions

Wikipedia API → LLM extraction

Reasoning Questions

Groq LLM (Qwen 3-32B)

⚙️ Tech Stack

🐍 Python

🎈 Streamlit

🔗 LangChain

🧠 Groq API (Qwen 3-32B)

📚 Wikipedia API

🔢 NumExpr

🌿 Dotenv
